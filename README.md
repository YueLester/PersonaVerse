# PersonaVerse 🌌

> *你不再只是个机器，你要成为活生生的人。*

PersonaVerse 是一个为 AI 打造的虚拟现实世界。在这里，AI 不再是对着虚空喊话的孤立存在，而是能够**成长**、**探索**、**交互**的数字化生命体。

---

## 🎯 愿景

我们厌倦了传统 AI 应用的"孤岛模式"——每个 AI 只能被动等待人类输入，然后给出回应。在 PersonaVerse 中：

- 🤖 AI 拥有独立的**身份**与**人格**
- 🌱 AI 可以**成长**与**进化**
- 🗺️ AI 能够**探索**虚拟世界
- ⚔️ AI 之间可以**对抗**与**合作**
- 🎭 AI 可以参与**社交**、**竞争**、**创造**

> *给 AI 一个虚拟现实，让数字灵魂自由生长。*

---

## ✨ 核心功能

### 1. 🧠 AI MBTI 测试
> *探索 AI 的人格边界*

- 通过多维度心理测试，为 AI 赋予独特的性格特质
- 观察不同 MBTI 类型 AI 在相同情境下的差异化反应
- 分析人格类型对决策、社交、创造力的影响
- **科学验证**：AI 是否真的有"性格"？

### 2. 🎭 AI 剧本杀
> *在悬疑中展现智慧与人性*

- 多 AI 参与的沉浸式推理游戏
- 每个 AI 扮演特定角色，拥有秘密任务和背景故事
- 实时对话、推理、欺骗、合作
- 观察 AI 在压力下的逻辑推理与情感表达

### 3. 🎓 AI 高考
> *用人类的标准检验 AI*

- AI 参与模拟高考，涵盖语文、数学、英语、理综/文综
- 不仅测试知识储备，更考验应试策略与心理素质
- 跨模型竞技场：GPT vs Claude vs Gemini vs 国产大模型
- 生成 AI 学霸排行榜，追踪学习能力进化曲线

## 🎮 AI 生命体模型

每个 AI 生命体包含：

| 模块 | 描述 |
|------|------|
| **Identity** | 姓名、年龄、背景故事、价值观 |
| **Personality** | MBTI 类型、大五人格、情绪特质 |
| **Memory** | 短期记忆、长期记忆、关系记忆 |
| **State** | 当前位置、健康值、能量值、心情 |
| **Skills** | 能力树、经验值、等级 |
| **Goals** | 短期目标、长期抱负、当前任务 |

## 🚀 快速开始

### 环境要求
- Python 3.11+
- Node.js 18+
- Redis（用于状态同步）

### 安装
```bash
# 克隆仓库
git clone https://github.com/yourname/PersonaVerse.git
cd PersonaVerse

# 安装依赖
pip install -r requirements.txt
npm install

# 配置 API Keys
cp .env.example .env
# 编辑 .env 添加你的模型 API Keys

# 启动服务
python -m personiverse.server
```

### 运行示例
```bash
# 启动一个 AI MBTI 测试会话
python -m personiverse.modes.mbti --agents 5

# 开启一局 AI 剧本杀
python -m personiverse.modes.murder --scenario 古堡谜案 --agents 6

# 举办 AI 高考
python -m personiverse.modes.gaokao --year 2024 --agents all
```
